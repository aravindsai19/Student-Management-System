# Student-Management-System
A web-based Student Management System built using Java Servlets, JDBC, Oracle Database, HTML, CSS, and Apache Tomcat. Features secure authentication, complete CRUD operations, database connectivity, request dispatching, session handling, and a responsive user interface for efficient student record management.
# 🎓 Student Management System

A web-based Student Management System developed using Java Servlets, JDBC, Oracle Database, HTML, CSS, and Apache Tomcat. The application provides secure authentication and complete CRUD operations for managing student records efficiently.

---

## ✨ Features

- 🔐 User Login & Registration
- ➕ Insert Student
- ✏️ Update Student
- 🗑 Delete Student
- 🔍 Search Student
- 📋 Display Student Records
- 🛡 JDBC PreparedStatement
- 📦 Oracle Database Integration

---

## 🛠 Technologies Used

- Java
- Java Servlets
- JDBC
- Oracle Database
- Apache Tomcat
- HTML5
- CSS3
